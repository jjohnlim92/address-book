Welcome to Address Book API

Request format should be in JSON

Endpoints:

To query:

GET /contact?pageSize={}&page={}&query={}

To add a contact:

POST /contact

To get a contact:

GET /contact/{name}

To update a contact:

PUT /contact/{name}

To delete a contact:

DELETE /contact/{name}

Allowed fields:

name, first_name, middle_name, last_name, email_address, website, home_phone, work_phone, mobile_phone, address, notes
